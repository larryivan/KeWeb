## About the Lab

We are a research group led by Dr. Ke Wang at the School of Life Sciences, Fudan University. Our research focuses on human evolutionary history through the lenses of paleogenomics, bioinformatics, and population genetics.

Key research areas:
- **Paleogenomics**: Ancient DNA studies reconstructing population interactions over the past 10,000 years
- **Bioinformatics**: Development of computational tools using coalescent theory and site-frequency spectrum
- **Interdisciplinary Studies**: Integrating genetic evidence with archaeological hypotheses and linguistic theories

## Tech Stack

- [VitePress](https://vitepress.dev/) — Static site generator
- [Vue 3](https://vuejs.org/) — UI framework
- [Tailwind CSS](https://tailwindcss.com/) — Styling
- [TypeScript](https://www.typescriptlang.org/) — Type safety

## Getting Started

```bash
# Install dependencies
pnpm install

# Start dev server
pnpm docs:dev

# Build for production
pnpm docs:build
```
